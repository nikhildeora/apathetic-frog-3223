# To Run Locally
## Clone the project
```bash
  git clone https://github.com/nikhildeora/fitnexy.git
```

Go to the project directory

```bash
  cd fitnexy
```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run start
```
