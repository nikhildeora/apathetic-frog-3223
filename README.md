# To Run Locally
## Clone the project
```bash
  git clone https://github.com/nikhildeora/apathetic-frog-3223.git
```

Go to the project directory

```bash
  cd apathetic-frog-3223
```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run start
```
